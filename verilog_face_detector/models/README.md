# Emotion Classification Model

This directory should contain the pre-trained emotion classification model.
The model should be named `emotion_mini_xception.h5` and placed in this directory.

You can train your own model or download a pre-trained one.
A popular choice for facial emotion recognition is the Mini-Xception model trained on the FER2013 dataset.

**Note:** The `download_model.py` script is a placeholder. You will need to implement it or manually download the model.
